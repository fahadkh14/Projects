# 🚀 Docker Task Manager (Production-like System)

## 📌 Overview
This is a containerized full-stack Task Manager app built using Docker.  
It simulates a real company micro-service setup with frontend, backend, and database.

---

## 🏗️ Architecture

Frontend (Nginx) → Backend API (Node.js) → MongoDB Database

---

## ⚙️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Database: MongoDB
- Containerization: Docker, Docker Compose
